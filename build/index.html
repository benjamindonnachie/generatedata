User documentation

Welcome to the generatedata.com user documentation! The purpose of this section is to explain what the application does, how to install it, and provide a little history on the script.

Note that the production website for this script (generatedata.com) is a private fork of this repo. They're very similar, but the public site contains a few additions that you won't see in this script, such as the payment registration options. See the website doc page for more info on that.
